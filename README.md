## Usage
```sh
git clone https://github.com/TylorS/lib-starter LIBRARY_NAME

cd LIBRARY_NAME

rm -rf .git

git init

git remote add origin git@github.com:USERNAME/REPO_NAME

node .scripts/addDevDependencies.sh
```

- Edit the first 3 fields in the package.json
- Develop!
